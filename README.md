# Quest

# 1. Project Overview

This project involves developing an interactive educational game using the Pygame library. The aim 
of the game is to enable users to learn or revise knowledge in a 
fun way, in the form of visual quizzes. Players must correctly match elements (images 
words, numbers) in an animated and user-friendly graphical environment.


# 2. Setup & Execution

**Prerequisites**
* Python 3.10+ 
* pip (package manager)


**Installation**

**1. Clone the repository**

`git clone https://github.com/Anthony10012/Quest-ICT.git`

**2. Setup the project**

`pip install -r requirements.txt`

**3. Run the game**

`cd src
python main.py`

# 3. How to play
1. Enter your username on the login screen

2. Use the menu to choose between starting a Quiz or viewing your Statistics

3. Answers the questions
